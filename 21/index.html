Merge Two Sorted Lists

You are given the heads of two sorted linked lists list1 and list2.

Merge the two lists in a one sorted list. The list should be made by splicing together the nodes of the first two lists.

Return the head of the merged linked list.

Example 1:

Input: list1 = [1,2,4], list2 = [1,3,4] Output: [1,1,2,3,4,4]

Example 2:

Input: list1 = [], list2 = [] Output: []

Example 3:

Input: list1 = [], list2 = [0] Output: [0]

Constraints:

  • The number of nodes in both lists is in the range [0, 50].
  • -100 <= Node.val <= 100
  • Both list1 and list2 are sorted in non-decreasing order.

impl Solution {
    pub fn merge_two_lists(list1: Option>, list2: Option>) -> Option> {
        match (list1, list2) {
            (None, None) => None,
            (None, Some(node)) => Some(node),
            (Some(node), None) => Some(node),
            (Some(node1), Some(node2)) => {
                let mut head = if node1.val < node2.val {
                    node1
                } else {
                    node2
                };
                let mut tail = head.clone();
                let mut next1 = node1.next.clone();
                let mut next2 = node2.next.clone();
                while let (Some(node1), Some(node2)) = (next1.as_mut(), next2.as_mut()) {
                    if node1.val < node2.val {
                        tail.next = Some(node1.clone());
                        tail = node1.clone();
                        next1 = node1.next.clone();
                    } else {
                        tail.next = Some(node2.clone());
                        tail = node2.clone();
                        next2 = node2.next.clone();
                    }
                }
                tail.next = next1.or(next2);
                Some(head)
            }
        }
    }

class Solution:
    def mergeTwoLists(self, list1: ListNode, list2: ListNode) -> ListNode:
        dummy = ListNode(0)
        tail = dummy
        while list1 and list2:
            if list1.val <= list2.val:
                tail.next = list1
                list1 = list1.next
            else:
                tail.next = list2
                list2 = list2.next
