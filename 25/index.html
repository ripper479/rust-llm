Reverse Nodes in k-Group

üè† ‚¨ÖÔ∏è ‚û°Ô∏è

Given the head of a linked list, reverse the nodes of the list k at a time, and return the modified list.

k is a positive integer and is less than or equal to the length of the linked list. If the number of nodes is not a multiple of k then left-out nodes, in the end, should remain as it is.

You may not alter the values in the list's nodes, only nodes themselves may be changed.

Example 1:

Input: head = [1,2,3,4,5], k = 2 Output: [2,1,4,3,5]

Example 2:

Input: head = [1,2,3,4,5], k = 3 Output: [3,2,1,4,5]

Constraints:

  • The number of nodes in the list is n.
  • 1 <= k <= n <= 5000
  • 0 <= Node.val <= 1000

Follow-up: Can you solve the problem in O(1) extra memory space?

impl Solution {
    pub fn reverse_k_group(mut head: Option>, k: i32) -> Option> {
        let mut count = 0;
        let mut current = &mut head;
        while let Some(node) = current {
            count += 1;
            current = &mut node.next;
        }

        let mut groups = count / k;
        let mut tail = head.as_mut();
        let mut prev = None;
        while let Some(node) = tail {
            let mut next = node.next.take();
            let mut current = node;
            for _ in 0..k {
                let next_node = next.take();
                current.next = next_node;
                current = next_node.unwrap();
            }
            current.next = prev;
            prev = Some(node);
            tail = current.next.as_mut();
            groups -= 1;
            if groups == 0 {
                break;
            }
        }

        head
    }
}

use std::option::Option;

struct ListNode {
    val: i32,
    next: Option>,
}

impl ListNode {
    fn new(val: i32) -> Self {
        ListNode {
            next: None,
            val,
        }
    }
}

struct Solution;

impl Solution {
    pub fn reverse_k_group(head: Option>, k: i32) -> Option> {
        let mut current = head;
        let mut previous = None;
        let mut count = 0;

        while let Some(node) = current {
            count += 1;
            if count == k {
                break;
            }
            current = node.next;
        }

        if count < k {
            return head;
        }

        let mut new_head = current;
        let mut new_tail = None;

        while let Some(node) = current {
            let mut next = node.next;
            node.next = previous;
            previous = Some(node);
            current = next;
        }

        new_head.next = previous;
        new_tail = new_head;

        while let Some(node) = new_tail {
            if node.next.is_none() {
                break;
            }
            new_tail = node.next;
        }

        new_tail.next = None;

        new_head
    }
}
